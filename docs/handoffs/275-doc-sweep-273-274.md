## SKILL.md

Add to the conventions:

- Hearing meeting titles are cleaned at render time via `cleanMeetingTitle()` in `lib/hearings.ts`. Deterministic string rules strip Senate procedural boilerplate ("Hearings to examine…", "Business meeting to consider…", "To receive a closed briefing on…", markups, oversight/legislative hearings), drop a leftover leading "the", re-capitalize, and fall back to the raw title if a strip empties it. Raw titles stay in the DB. All three render sites import the one helper: HearingRow, HearingsCalendar, OnTheHillBand. Don't re-clean at ingest and don't duplicate the regex.
- Competitive race-card market odds (Kalshi + Polymarket) render as party lean, not candidate name, so the K/P pair is comparable. RaceCard resolves a candidate-named market to a party using the seat's full candidate roster (prefetched in the hub), not the incumbent alone; resolving from the incumbent only is the bug that left challenger markets showing a surname. House seats with no Polymarket collapse to the Kalshi cell. Fall back to the candidate name only if a candidate can't be mapped to a party.
- Battlefield markers and race-card candidate dots both color from the seat's incumbent party (`members.party` via the incumbent join), which keeps the two surfaces consistent. `getBattlefieldSeats.incumbentParty` -> `partyColor(...)`.
- The v2 markets/signals tapes pin the MARKETS / SIGNALS strip label on the left, outside the scrolling marquee (mirror of the right-pinned AS OF meta: absolute, opaque, fade-edged). Only the ticks scroll. The root tape and inner-page tapes stay label-free and static.

## docs/oddities.md

- Committee meeting titles can carry a non-breaking space (U+00A0) or narrow NBSP between words (e.g. between "examine" and the topic), which silently defeats a literal-space prefix strip. Normalize NBSP on input before any prefix matching.
- FRED commodity/rate series lag in reporting (DCOILWTICO confirmed; assume others do too). The markets cron can diff a fresh value against a stale prior row when the intervening sessions were never captured, producing a phantom large daily move (the WTI -10.89% case diffed against a week-old 95.0). A ±8% render guard drops the arrow and change on any implausible daily move while keeping the real price, which masks the egregious ones.
- Battlefield marker position is rating-based (lean score) while race-card odds are market-based, so the two can point different ways for the same seat (ME-SEN sits toward R on ratings while the markets favor D). Expected, not a bug. Don't "fix" it.
