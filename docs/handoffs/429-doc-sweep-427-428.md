### 2. `.claude/skills/cbt/SKILL.md` — schema (after the `member_ideology` CREATE TABLE, ~line 664+)

Add the table verbatim from `migrate.ts`:

```sql
CREATE TABLE polarization_history (
  congress    INTEGER NOT NULL,
  chamber     TEXT    NOT NULL,   -- normalized lowercase: 'house' | 'senate'
  dem_median  REAL,               -- median nominate_dim1 over party_code 100
  rep_median  REAL,               -- median nominate_dim1 over party_code 200
  PRIMARY KEY (congress, chamber)
);
```

One line of prose: per-Congress/per-chamber D/R medians of DW-NOMINATE `dim1`, aggregated from `HSall`. `gap` is **not stored** — derived as `|rep_median − dem_median|` at read time (the identical definition the band uses; one gap formula in the codebase). ~a few hundred rows; PK covers every read, no secondary index.
