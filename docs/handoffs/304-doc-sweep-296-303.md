- The v2 feed queries (getStageChanges / getStaleBills / getNewBillsThisWeek) now carry the full sponsor enrichment (the getFeedBills SPONSOR_ENRICH select/join: depiction_url, first_name, last_name, district, cosponsor_count). Without it the sponsor card degrades (raw "Last, First" name, wrong district, initials instead of photo, no cosponsors). The INDEXED BY hints on those queries are untouched.

The hover box is the four-part box: name (amber-bright) + sector / note (muted, omit-when-empty) / figure (markets = white value + colored change; odds = K/P brand letters + white numbers; CPI/UNEMP = value alone) / freshness line (real per-item: EOD for FRED-daily, time for intraday, "FRED · monthly · released [month]" for CPI/UNEMP, "Live · prediction market" for odds).
- lib/policy-hooks.ts is now `Record<string,{sector,note?}>` (curated sector + optional note per item, keyed by internal symbol).
- MarketSymbol gained `kalshiEvent` (pin a specific Kalshi event instead of series-soonest) and `polyMonth` (pin the Polymarket fed-decision month), used to carry distinct same-series markets like the July and September Fed decisions.
