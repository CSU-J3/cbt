# 93.5 — Louisiana House primaries (closed partisan, not jungle)

## What this is

Closes the LA carve-out from HO 93. The shape changed: Louisiana switched congressional races from the open "jungle" system to closed partisan primaries starting in 2026. There is no jungle parser to write — LA House is the same closed partisan format as the rest of the South. This handoff adds the 6 LA House districts to the South region's target list and lets the existing partisan parser pick them up via the HO 97 cursor.

Two complications to verify before coding:

1. **The Senate side may already be captured.** HO 91 scraped LA Senate. The May 16 R primary (Cassidy → Letlow + Fleming runoff) used a Ballotpedia URL with a special suffix: `United_States_Senate_election_in_Louisiana,_2026_(May_16_Republican_primary)`. If HO 91 hit the standard URL pattern, it may have missed this. Verify what's in the DB.
2. **LA House primaries were suspended mid-cycle.** Governor Landry voided the May 16 House primary results after ~40,000 absentee ballots were cast. Ballotpedia House district pages may reflect the old map, the new rescheduled dates, or be sparse. Spot-check before assuming standard URL patterns work.

## Pre-flight verification (mandatory)

Before writing any code:

1. Fetch `https://ballotpedia.org/Louisiana%27s_1st_Congressional_District_election,_2026` — does the page exist and have candidate sections?
2. Same for LA-02 through LA-06.
3. Check the LA Senate page in the DB: `SELECT * FROM primaries WHERE id LIKE 'senate-LA-2026%'` — is the May 16 Cassidy race captured? If not, the Senate gap is a separate item to fix (see "Out of scope").
4. If any LA House page returns a redirect, "Election postponed" placeholder, or empty roster, log the per-district URL + status and proceed with whichever districts have data. Don't block the handoff on the disrupted ones — partial coverage is acceptable since the cursor will pick them up automatically once Ballotpedia updates.

## Background context (worth knowing, don't bake into code)

- LA switched US Senate, US House, LA Supreme Court, Public Service Commission, and BESE seats to closed partisan primaries starting 2026. Governor and state legislature remain jungle (next contested 2027).
- Primary system: closed R + closed D primaries, semi-open in practice (unaffiliated voters can pick one party's ballot, members of recognized minor parties can't). If no >50% winner in either primary, top-two-within-party runoff. The Cassidy → Letlow/Fleming June 27 runoff is the canonical example.
- The 2024 redistricting added a second majority-Black district (Robinson v. Ardoin / VRA outcome). LA still has 6 House seats; the map is just redrawn. District numbering 1-6 still applies.
- LA Senate's incumbent matchup: Cassidy lost his primary on May 16. Whoever wins the R runoff (Letlow vs. Fleming, June 27) faces the D nominee in November. The incumbent matcher will need to handle Cassidy as "lost primary" status downstream — out of scope here.

## Implementation

After pre-flight passes:

```ts
// In whatever file holds HOUSE_DISTRICTS_SOUTH_2026
export const HOUSE_DISTRICTS_SOUTH_2026 = [
  // ... existing TX, GA, FL, etc.
  { state: 'LA', district: 1 },
  { state: 'LA', district: 2 },
  { state: 'LA', district: 3 },
  { state: 'LA', district: 4 },
  { state: 'LA', district: 5 },
  { state: 'LA', district: 6 },
];
```

The target list `buildTargetList()` rebuilds automatically. The cursor's `total_targets` field updates on next tick. No migration needed.

To catch up immediately rather than waiting on the cursor:

```powershell
npm run sync:house-primaries -- --region=south
```

This re-scrapes the whole South region including LA. The HTML cache (from HO 92) makes existing-state re-scrapes cheap; the 6 new LA pages are the only cold fetches.
